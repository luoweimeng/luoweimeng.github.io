seq2seq

Posted on 2018-05-02

Deep neural network：

\[H = \phi(XW_{xh} + b_n)\]

Assume the length of hidden layer is \(h\), input is\(X\in \mathbb{R}^{n \times x}\), weight is \(W_{xh}\in \mathbb{R}^{x \times h}\), bias is\(b\in \mathbb{R}^{1 \times h}\). The output of hidden layer is \(H \in \mathbb{R}^{n \times h}\)

Use the output of hiidden layer (hidden state) as the input to the output layer (a dense layer followed by the softmax operation), we could get the output \(Y\):

\[Y = \text{softmax}(HW_{hy}+b_y) \]

RNN(Recurrent neural network)：

将上面的网络改为RNN，我们首先对输入输出加入时间戳。现在的输入为\(X_t\in \mathbb{R}^{n \times x}\)，即序列中第\(t\)个批量输入（样本数为n，feature数为x），对应的隐含层输出为\(H_t \in \mathbb{R}^{n \times h}\)。现在在计算\(H_t\)时，我们不仅需要考虑\(X_t\)，还需要考虑前一个时刻的隐含层输出\(H_{t-1}\)。故而新的隐含层计算公式为：

\[H_t = \phi(X_tW_{xh} + H_{t-1}W_{hh} + b_n)\]

最终\(t\)时刻的输出\(Y_t\)与前面一致：

\[Y_t = \text{softmax}(H_tW_{hy}+b_y) \]

现在，新的隐含状态\(H_t\)与之前所有时刻的输入\(X_t', t' \in\{1,2,...,t-1\}\)有关。故而可以说，隐含状态\(H_t\)可以认为是这个网络的记忆，它储存了前面时刻的信息。

LSTM(Long Short-Term Memory):

Input gate, forget gate, output gate

To solve the problem of vinishing gradient and better capture long-term dependencies of time series, a network called long short-term memory is proposed. The input of LSTM is same as the vanilla RNN, which is the current input \(X_t\in \mathbb{R}^{n \times x}\), and the previous hidden state \(H_{t-1} \in \mathbb{R}^{n \times h}\).

We can calculate the input gate \(I_t \in \mathbb{R}^{n \times h}\), forget gate \(F_t \in \mathbb{R}^{n \times h}\), output gate \(O_t \in \mathbb{R}^{n \times h}\) by following equations:

\[I_t = \sigma(X_tW_{xi}+H_{t-1}W_{hi}+b_i\] \[F_t = \sigma(X_tW_{xf}+H_{t-1}W_{hf}+b_f\] \[O_t = \sigma(X_tW_{xo}+H_{t-1}W_{ho}+b_o)\]

LSTM使用cell state来传递信息，并且可以通过门的机制来加入或删除信息。具体来说，sigmoid函数可以用来控制信息传递，0意味着不让信息通过，1意味着信息可以通过。

Candidate cell

Candidate cell \(\tilde{C} \in \mathbb{R}^{n \times h}\) 使用了值域在[-1,1]的双曲正切函数tanh做激活函数

\[\tilde{C}=tanh(X_tW_{xc}+H_{t-1}W_{hc}+b_c\]

Cell state

We use the input gate and forget gate to control the information flow between cell states.

\[C_t=F_t \odot C_{t-1} + I_t \odot \tilde{C}_t\]

Hidden state

Finally, we use output gate to control the information flow from cell state to hidden state.

\[H_t=O_t \odot tanh(C_t) \]
